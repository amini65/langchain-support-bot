# LangChain Customer Support Bot

A production-ready customer support chatbot built with LangChain, featuring OpenAI function calling, conversation memory, and custom tools.

## 🚀 Features

- **Intelligent Agent**: Uses OpenAI's function calling to decide which tool to use
- **Product Database**: Query product prices and list all available items
- **Conversation Memory**: Maintains chat history throughout the session
- **Email Integration**: Sends conversation summaries to admin
- **Clean Architecture**: Modular design following best practices
